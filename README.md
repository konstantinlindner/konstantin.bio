# konstantin.bio

This is the repository for my personal "link in bio", also referred to as "link tree" - page.

## Stack

Language: Typescript  
Framework: React
CSS: Chakra UI  
Build tool: Vite  
Icons: Iconify

## Setup

Prerequisites: node, yarn.

Clone and enter the project, then run:

```bash
$ yarn install
$ yarn dev # serves at localhost:3000
```
